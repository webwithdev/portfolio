About Devansh

I build end-to-end web products with a focus on performance, clean architecture, and real-world usability. From UI design to database modeling and deployment, everything stays aligned with the product goal.

My stack centers around React and Next.js on the frontend, Node.js and MongoDB on the backend, and Vercel for smooth deployments. I care a lot about DX, file structure, reusable components, and clear API contracts.

Frontend

Modern, responsive interfaces with React, Next.js, Tailwind CSS, and component-driven design.

Backend & Infra

Node.js, Express, MongoDB, and clean REST APIs deployed with GitHub + Vercel workflows.
